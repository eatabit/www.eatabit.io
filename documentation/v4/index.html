Authentication

API endpoint
https://api.eatabit.io
SSL
All requests must be made over HTTPS.
Authentication
All requests are authenticated with HTTP Basic auth. The username is your account :sid and the password is your account :token. You can supply a DEVELOPMENT or PRODUCTION account :sid & :token pair. Jobs created with a DEVELOPMENT account will not actually be sent to a printer.

Valid request

Verb GET
Format
https://api.eatabit.io/v2/account/:sid.json
Example
curl https://api.eatabit.io/v2/account/617117b5-4042-4951-b75c-88f2ee159afd.json \
-X GET \
-u 617117b5-4042-4951-b75c-88f2ee159afd:c7ae2067-95bd-4722-909a-9307ce019097
Response code 200
Response body
JSON: {
"account" : {
"name" : "eatabit.io",
"support_phone_number" : "+15551234567",
"created" : "2013-11-27T15:26:43Z"
}
}
