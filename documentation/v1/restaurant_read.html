Restaurant READ

/v1/account/:account_id/restaurants/:restaurant_id.json

Valid request

Verb GET
Example
curl https://api.eatabit.io/v1/account/839df546-cff4-4b41-999e-6cdfa6fd2e78/restaurants/+15551234567.json \
-X GET \
-u 839df546-cff4-4b41-999e-6cdfa6fd2e78:M4FmfvvHCCb5LmcLHWRcQg
Response code 200
Response body
JSON: {
"object" : "restaurant",
"name" : "Bill's Pizza",
"enabled" : true,
"premise_phone_number" : "+15551234567",
"printer_state" : "online",
"pickup_minutes" : 15,
"delivery_minutes" : 30,
"created" : "2013-11-27T15:26:43Z"
}

Request route parameters

:account_id
Position: /v1/account/:account_id/restaurants/+15551234567.json
Description: The UUID of your eatabit.io account. This is the same UUID as is passed as USER in the request HTTP BASIC AUTH.
:restaurant_id
Position: /v1/account/839df546-cff4-4b41-999e-6cdfa6fd2e78/restaurants/:restaurant_id.json
Description: The restaurant premise phone number in E.164 format (including the leading '+').

Response parameters

object
Type: String
Value: "restaurant"
Description: The class of the returned object. Provided for convenience.
name
Type: String
Description: The name of the restaurant.
enabled
Type: Boolean
Description: The administrative state of the restauant. Restaurants that are disabled cannot receive orders.
premise_phone_number
Type: String
Description: The phone number of the restaurant in E.164 format.
printer_state
Type: String
Description: The current state of the printer. Values are 'online' or 'offline'. If the printer_state equals 'offline', the API will not accept orders for that printer until it comes back online.
pickup_minutes
Type: Integer
Description: The current fulfillment time for pickup orders, in minutes.
delivery_minutes
Type: Integer
Description: The current fulfillment time for delivery orders, in minutes.
created
Type: Timestamp
Description: The UTC time that the record was created. Timestamps are in iso8601 format.
