Authentication

API endpoint
https://api.eatabit.io
SSL
All requests must be made over HTTPS.
Authentication
All requests are authenticated with HTTP Basic auth. The username is your :account_id and the password is your key. You can supply a DEVELOPMENT or PRODUCTION key. Orders created with a development key will not be sent to a printer.

Valid request

Verb GET
Example
curl https://api.eatabit.io/v1/account/ad1b5852-d22f-49c8-8b09-31995ac519a3.json \
-X GET \
-u ad1b5852-d22f-49c8-8b09-31995ac519a3:6s3AcHtv4Eja2ZkEhtPo4w
Response code 200
Response body
JSON: {
"object" : "account",
"name" : "eatabit.io",
"support_phone_number" : "+15551234567",
"created" : "2013-11-27T15:26:43Z"
}
