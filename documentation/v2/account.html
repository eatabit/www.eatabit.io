READ

/v2/account/:sid.json

Valid request

Verb GET
Format
https://api.eatabit.io/:api_version/account/:sid.json
Example
curl https://api.eatabit.io/v2/account/839df546-cff4-4b41-999e-6cdfa6fd2e78.json \
-X GET \
-u 839df546-cff4-4b41-999e-6cdfa6fd2e78:M4FmfvvHCCb5LmcLHWRcQg
Response code 200
Response body
JSON: {
"account" : {
"name" : "eatabit.io",
"support_phone_number" : "+15551234567",
"created" : "2013-11-27T15:26:43Z"
}
}

Request route parameters

:api_version
Position: https://api.eatabit.io/:api_version/account/839df546-cff4-4b41-999e-6cdfa6fd2e78.json
Description: The :api_version of the eatabit.io API. The current version is v2.
:sid
Position: https://api.eatabit.io/v2/account/:sid.json
Description: The :sid of your eatabit.io account. This is the same sid as is passed as USER in the request HTTP Basic auth.

Response parameters

:name
Type: String
Description: Your account's name. This string will appear on printed receipts.
:support_phone_number
Type: String
Description: Your account's support phone number in E.164 format.
:created
Type: Timestamp
Description: The UTC time that the record was created. Timestamps are in iso_8601 format.
