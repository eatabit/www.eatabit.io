Printer

READ

Valid request

Verb GET
Example
curl https://api.eatabit.io/v2/account/617117b5-4042-4951-b75c-88f2ee159afd/printers/fc4a764b-4822-45d5-b91f-bc808412002f.json \
-X GET \
-u 617117b5-4042-4951-b75c-88f2ee159afd:c7ae2067-95bd-4722-909a-9307ce019097

Response body
JSON: {
"printer" : {

"enabled" : true,
"pickup_minutes" : 15,
"delivery_minutes" : 30,

}
}

Request route parameters

:api_version
Position: https://api.eatabit.io/:api_version/account/617117b5-4042-4951-b75c-88f2ee159afd/printers/fc4a764b-4822-45d5-b91f-bc808412002f.json
Description: The :api_version of the eatabit.io API. The current version is: v2.
:sid
Position: https://api.eatabit.io/v2/account/:sid/printers/fc4a764b-4822-45d5-b91f-bc808412002f.json
Description: The :sid of your eatabit.io account. This is the same sid as is passed as USER in the request HTTP Basic auth.
:id
Position: https://api.eatabit.io/v2/account/617117b5-4042-4951-b75c-88f2ee159afd/printers/:id.json
Description: The printer's id on the eatabit.io API. Contact us

Response parameters

name
Type: String
Description: The name of the Restaurant that the printer is located in.
enabled
Type: Boolean
Description: The administrative state of the restauant. Restaurants that are disabled cannot receive orders.
state
Type: String
Description: The current state of the printer. Values are 'online' or 'offline'. If the printer_state equals 'offline', the API will not accept orders for that printer until it comes back online.
pickup_minutes
Type: Integer
Description: The current fulfillment time for pickup orders, in minutes.
delivery_minutes
Type: Integer
Description: The current fulfillment time for delivery orders, in minutes.
