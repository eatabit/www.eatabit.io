Request route parameters

:api_version
Position: https://api.eatabit.io/:api_version/account/617117b5-4042-4951-b75c-88f2ee159afd/printers/fc4a764b-4822-45d5-b91f-bc808412002f/jobs/9696.json
Description: The :api_version of the eatabit.io API. The current version is v2.
:sid
Position: https://api.eatabit.io/v2/account/:sid/printers/fc4a764b-4822-45d5-b91f-bc808412002f/jobs/9696.json
Description: The :sid of your eatabit.io account. This is the same sid as is passed as USER in the request HTTP Basic auth.
:restaurant_id
Position: https://api.eatabit.io/v2/account/617117b5-4042-4951-b75c-88f2ee159afd/printers/:printer_id/jobs/9696.json
Description: The printer's id on the eatabit.io API. Contact us
:id
Position: https://api.eatabit.io/v2/account/617117b5-4042-4951-b75c-88f2ee159afd/printers/fc4a764b-4822-45d5-b91f-bc808412002f/jobs/:id.json
Description: The ID of the job in the eatabit database. It is returned in a successful job CREATE response.

Response parameters

id
Type: Integer
Description: The ID of the job in the eatabit database. It is returned in a successful job CREATE response.
body
Type: String
Description: The body of the job to be printed.
state
Type: String
Description: The current state of the job. Possible states include:
queued
The job is queued to be downloaded to the printer.
downloaded
The job has been downloaded to the printer.
printed
The job has been printed by the printer.
paused
The job has been paused by the creator.
expired
The job has not been printed within 5 minutes of being downloaded.
cancelled
The job has been cancelled by the creator.
status_url
Type: String
Description: The status_url specified when the job was created. This url will be actioned every time the job state changes.
environment
Type: String
Description: The job environment as defined by the account used to create it. Jobs created using your development account will be of environment: 'development'. Development jobs are for testing purposes and will not be sent to the printer.
created
Type: Timestamp
Description: The UTC time that the record was created. Timestamps are in iso8601 format.
