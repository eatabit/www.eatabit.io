Job

CREATE

/v2/account/:sid/restaurants/:restaurant_id/jobs.json

Valid request

Verb POST
Example
curl https://api.eatabit.io/v2/account/617117b5-4042-4951-b75c-88f2ee159afd/restaurants/+15551234567/jobs.json \
-X POST \
-u 617117b5-4042-4951-b75c-88f2ee159afd:c7ae2067-95bd-4722-909a-9307ce019097 \
--data-urlencode 'body=One large pepperoni pizza with breadsticks'
Response code 200
Response body
JSON: {
"job" : {
"id" : 2048,
"body" : "One large pepperoni pizza",
"state" : "queued",
"created" : "2013-11-27T15:26:43Z"
}
}

Request route parameters

:api_version
Position: https://api.eatabit.io/:api_version/account/617117b5-4042-4951-b75c-88f2ee159afd/restaurants/+15551234567/jobs.json
Description: The :api_version of the eatabit.io API. The current version is v2.
:sid
Position: https://api.eatabit.io/v2/account/:sid/restaurants/+15551234567/jobs.json
Description: The :sid of your eatabit.io account. This is the same sid as is passed as USER in the request HTTP Basic auth.
:restaurant_id
Position: https://api.eatabit.io/v2/account/617117b5-4042-4951-b75c-88f2ee159afd/restaurants/:restaurant_id/jobs.json
Description: The restaurant premise phone number in E.164 format (including the leading '+').

Request parameters

body
Required: Yes
Type: String
Description: The text to be printed on the paper ticket.

Response parameters

id
Type: Integer
Description: The ID of the job in the eatabit database.
body
Type: String
Description: The body of the job to be printed.
state
Type: String
Description: The current state of the job. Possible states include:
queued
The job is queued to be downloaded to the printer.
downloaded
The job has been downloaded to the printer.
printed
The job has been printed by the printer.
paused
The job has been paused by the creator.
expired
The job has not been printed within 5 minutes of being downloaded.
cancelled
The job has been cancelled by the creator.
environment
Type: String
Description: The job environment as defined by the account used to create it. Jobs created using your development account will be of environment: 'development'. Development jobs are for testing purposes and will not be sent to the printer.
created
Type: Timestamp
Description: The UTC time that the record was created. Timestamps are in iso8601 format.
